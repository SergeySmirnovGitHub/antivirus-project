# done — готовые сборки

Сюда складываются **готовые к запуску** версии программ (например, `vtscan.exe`),
собранные из исходников в соседних папках `apps/<программа>/`.

## Как здесь появляется vtscan.exe

`.exe` — формат Windows и собирается **только на Windows-машине** (кросс-сборки с Mac нет).

1. Скопируй проект на Windows-машину (или открой его там).
2. Установи Python 3.10+ с https://www.python.org/ (при установке галочка **«Add Python to PATH»**).
3. Зайди в папку `apps/vtscan/` и запусти **двойным кликом** файл `build_windows.bat`.
4. Скрипт сам поставит зависимости и соберёт `vtscan.exe` прямо сюда, в `apps/done/`.

## Как пользоваться готовым vtscan.exe

Рядом с `vtscan.exe` положи файл `.vtkey` со своим ключом VirusTotal
(или задай переменную окружения `VT_API_KEY`). Затем в командной строке:

```
vtscan.exe C:\путь\к\файлу.exe
vtscan.exe C:\Users\Имя\Downloads -r
```
